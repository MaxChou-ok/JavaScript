### 正则
含义：专门用来检索字符串的一种规则(更擅长处理模糊范围的字符串)
字面量写法：
new RegExp 
```
1. test()方法检索字符串中的指定值。返回值是 true 或 false。
2. exec()找到正则匹配的字符串（首次出现的字符【就一次】），并且放到数组中(返回的是数组(要找的东西,索引，全部的，undefined))
3. \ 转义字符   把正则中有特殊含义的字符 转成字符本身(不再有任何特殊的含义),转义符转的是\后面的字符
4. 特殊转义符:\d  一个0-9的数字
5. \D 非数字
6. 量词: +  至少出现1次，最多不限
7. 修饰符: g -> global 全局查找
8. string.match(//)是字符串的方法,找到正则匹配的字符并且把它们放到数组中（返回值是数组）
 特性：懒惰性：让找几个找几个
       贪婪性：只要符合某个规则就不停的找，直到找不到为止
9. string.replace(''|正则,''|函数)
10.[] 中的字符在正则中是找任意一个字符
11.{} 描述{}前面字符的数量
12.? 最少可以没有，最多出现1次
13.* 最少可以没有，最多无限
14. ^ 如果^在中括号中，那么代表排除，代表以什么字符开头
15. $ ：代表以什么字符结尾
16. \n ：代表换行

补充：中文的区间范围:[\u4e00-\u9fa5]
```
```
   let str = 'abc';
   let aa = 'a';
    console.log(/aa/); 找的是字符aa(正则里面放的是不带引号的字符串或者规则)
    console.log(new RegExp(aa)); 找的是变量aa，其实就是找字符串a
```
